**lua/neat/genome.lua**: Genome and Gene data structures.
- `newGene()`: returns `{into=0, out=0, weight=0.0, enabled=true, innovation=0}`
- `copyGene(gene)`: shallow copy of gene table
- `newGenome(config)`: returns genome with `{genes={}, fitness=0, maxneuron=0, globalRank=0, mutationRates={...}, network=nil}`. mutationRates initialized from config defaults: connections, link, node, bias, step, disable, enable.
- `copyGenome(genome)`: deep copy including all genes and mutationRates (NOT network -- regenerated on demand)
- `basicGenome(config, innovation)`: creates minimal genome with Inputs*Outputs random connections. Each connection: into=input_neuron, out=MaxNodes+output_index, weight=math.random()*4-2, innovation=innovation.newInnovation(). This is the seed genome for population initialization.

**lua/neat/network.lua**: Neural network construction and evaluation.
- `generateNetwork(genome, config)`: builds network from genome.genes. Creates neurons table indexed by neuron ID. Input neurons: 1..Inputs. Output neurons: MaxNodes+1..MaxNodes+Outputs. Hidden neurons: discovered from genes. Each neuron: `{incoming={}, value=0.0}`. For each enabled gene: add {into=gene.into, weight=gene.weight} to neurons[gene.out].incoming.
- `evaluateNetwork(network, inputs, config)`: forward pass. Set input neuron values from inputs array. Iterate all neurons in topological order (or use the MarI/O pattern: iterate all neurons, compute value = sigmoid(sum(incoming[i].weight * neurons[incoming[i].into].value))). Use NEAT sigmoid: `2.0 / (1.0 + math.exp(-4.9 * x)) - 1.0`. Return array of output neuron values (indices MaxNodes+1..MaxNodes+Outputs).
- `sigmoid(x)`: the activation function.

**lua/neat/mutation.lua**: All mutation operators.
- `pointMutate(genome, config)`: for each gene, with PerturbChance perturb weight by StepSize * random, else randomize weight to random()*4-2.
- `linkMutate(genome, config, innovation, forceBias)`: add connection between two non-connected neurons. If forceBias, source is always the bias neuron (neuron Inputs). Otherwise, random source. Random target (non-input). Generate network first to find valid neurons. Get innovation number. Add new gene.
- `nodeMutate(genome, config, innovation)`: pick a random enabled gene, disable it, create two new genes with a new neuron in between. Gene1: same into, new neuron out, weight 1.0. Gene2: new neuron in, same out, same weight as original. Both get new innovation numbers. Increment genome.maxneuron.
- `enableDisableMutate(genome, enable)`: toggle a random gene's enabled state.
- `mutate(genome, config, innovation)`: apply all mutation operators with their configured probabilities. This is the main mutation entry point called per genome per generation.

**lua/neat/crossover.lua**: Crossover operator.
- `crossover(g1, g2, config)`: if g2.fitness > g1.fitness, swap so g1 is fitter. Create child genome. For matching innovation numbers: randomly pick gene from g1 or g2. For disjoint/excess: take from g1 (fitter parent). Copy all genes. Set child.maxneuron = max(g1.maxneuron, g2.maxneuron). Return child.

**lua/neat/species.lua**: Species management.
- `disjoint(genes1, genes2)`: count disjoint genes normalized by max genome size
- `weights(genes1, genes2)`: average weight difference of matching genes
- `sameSpecies(genome1, genome2, config)`: compatibility distance < DeltaThreshold
- `addToSpecies(genome, pool, config)`: find matching species or create new one
- `calculateAverageFitness(species)`: sum of fitnesses / count (fitness sharing)
- `cullSpecies(pool, cutToOne)`: remove bottom half of each species (or all but best if cutToOne)
- `removeStaleSpecies(pool, config)`: remove species where staleness > StaleSpecies (never remove the species with the global best genome)
- `removeWeakSpecies(pool)`: remove species whose share of offspring (proportional to average fitness) rounds to 0
- `adjustCompatibilityThreshold(pool, config)`: dynamic DeltaThreshold targeting TargetSpecies

**lua/neat/pool.lua**: Population management.
- `newPool(config, innovation)`: create pool with generation=0, species={}, maxFitness=0. Call basicGenome() PopulationSize times and addToSpecies each.
- `rankGlobally(pool)`: flatten all genomes, sort by fitness, assign globalRank 1..N
- `totalAverageFitness(pool)`: sum of species average fitnesses
- `breedChild(species, pool, config, innovation)`: CrossoverChance -> crossover two random genomes from species, else copy random genome. Then mutate child.
- `newGeneration(pool, config, innovation)`: cull species, remove stale, remove weak, rank globally, calculate offspring shares. Breed children. Preserve elite (best genome of each species copied unchanged). Clear old species, re-speciate with new genomes. Increment generation. Adjust compatibility threshold.

**lua/game/fitness.lua**: Multi-component fighting game fitness function.
- `local config = dofile("lua/neat/config.lua")` for TimeoutConstant and StallFrameThreshold
- Define weight constants (tunable):
  ```
  W_OFFENSE = 2.0       -- reward per HP of damage dealt
  W_DEFENSE = 1.5       -- penalty per HP of damage taken
  W_WIN_BONUS = 1000    -- bonus for winning the round
  W_LOSE_PENALTY = 500  -- penalty for losing the round
  W_STALL = 0.5         -- penalty per frame of stalling beyond threshold
  ```
- Define round result constants: `WIN = 1, LOSE = 2, DRAW = 3, IN_PROGRESS = 0`
- `calculateFitness(params)` where params is a table:
  ```
  {
    startP1HP, endP1HP,       -- bot health before/after
    startP2HP, endP2HP,       -- opponent health before/after
    roundResult,              -- WIN/LOSE/DRAW/IN_PROGRESS
    frameCount,               -- total frames elapsed
    lastDamageFrame,          -- frame of most recent damage dealt
    damageDealtByBot          -- damage explicitly dealt by bot actions (FIT-06)
  }
  ```
  Calculation:
  1. `damageDealt = params.startP2HP - params.endP2HP`
  2. `damageTaken = params.startP1HP - params.endP1HP`
  3. `fitness = damageDealt * W_OFFENSE` (FIT-01)
  4. `fitness = fitness - damageTaken * W_DEFENSE` (FIT-02)
  5. If roundResult == WIN: `fitness = fitness + W_WIN_BONUS` (FIT-03)
  6. If roundResult == LOSE: `fitness = fitness - W_LOSE_PENALTY` (FIT-04)
  7. Anti-stall: `stallFrames = frameCount - lastDamageFrame; if stallFrames > StallFrameThreshold then fitness = fitness - (stallFrames - StallFrameThreshold) * W_STALL end` (FIT-05)
  8. Self-destruction guard: if damageDealtByBot is provided and damageDealt > damageDealtByBot, only credit damageDealtByBot (FIT-06)
  9. Floor: `if fitness <= 0 then fitness = -1 end` (avoid 0 which MarI/O uses as "not evaluated")
  10. Return fitness

- `getWeights()`: returns the weight table (for logging/tuning)
- `getRoundConstants()`: returns WIN, LOSE, DRAW, IN_PROGRESS

The fitness module must work standalone (no BizHawk dependency) for testing. Memory reads happen in the training loop, not in this module.
