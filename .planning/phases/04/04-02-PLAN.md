    Task that checks fitness threshold and triggers continuation if not met.

      Script that compares win-rate against fitness-threshold. If below threshold, sends a POST to `http://el-training-continuation.default:8080` with JSON body containing run-id and resume-from (the latest-genome-path). If threshold met, logs "Training complete."
    - Follow the exact pattern from RESEARCH.md Pattern 3.

    **event-listener.yaml** (`k8s/tekton/triggers/event-listener.yaml`):
    - apiVersion: triggers.tekton.dev/v1beta1

    - serviceAccountName: `tekton-triggers-sa`
    - Single trigger referencing the binding and template.

    **trigger-template.yaml** (`k8s/tekton/triggers/trigger-template.yaml`):
    - apiVersion: triggers.tekton.dev/v1beta1
    - name: `training-continuation-template`
    - params: `run-id`, `resume-from`
    - resourcetemplates: Creates a new PipelineRun with:
