    **Update run-training.yaml** (`k8s/tekton/tasks/run-training.yaml`):
    Read the existing task from Plan 01. Add a second step `push-metrics` after the `train` step:
    - image: `curlimages/curl:latest`
    - Script that reads training output files from workspace (generation number, max fitness, avg fitness, species count) and pushes to Pushgateway:
      ```bash
      #!/bin/sh
      RUN_ID="$(params.run-id)"
      GENERATION=$(cat $(workspaces.workspace.path)/output/final-generation.txt)
      MAX_FITNESS=$(cat $(workspaces.workspace.path)/output/best-fitness.txt)
      AVG_FITNESS=$(cat $(workspaces.workspace.path)/output/avg-fitness.txt)
      SPECIES_COUNT=$(cat $(workspaces.workspace.path)/output/species-count.txt)

      cat <<EOF | curl --data-binary @- http://pushgateway-prometheus-pushgateway.default.svc:9091/metrics/job/neat-training/run_id/${RUN_ID}
      # HELP neat_generation Current NEAT generation number
      # TYPE neat_generation gauge
      neat_generation ${GENERATION}
      # HELP neat_fitness_max Highest fitness in current generation
      # TYPE neat_fitness_max gauge
      neat_fitness_max ${MAX_FITNESS}
      # HELP neat_fitness_avg Average fitness across population
      # TYPE neat_fitness_avg gauge
      neat_fitness_avg ${AVG_FITNESS}
      # HELP neat_species_count Number of active species
      # TYPE neat_species_count gauge
      neat_species_count ${SPECIES_COUNT}
      EOF
      ```
    - Also add `run-id` as a param to the run-training task (needed for Pushgateway grouping key). Update pipeline.yaml to pass run-id to run-training.
